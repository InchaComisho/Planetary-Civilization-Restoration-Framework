# Planetary Civilization Restoration Framework

[English README](README.md)

## 概要

**Planetary Civilization Restoration Framework** は、InchaComisho によって作成された公開リポジトリ群のためのマスター索引・ナビゲーションマップです。

このリポジトリは、新しい技術提案そのものではありません。惑星修復と文明設計に関する、アイデア、仮説、概念フレームワーク、リスクシナリオ、技術提案、ガバナンス提案、公開コミュニケーション、フィクション/ナラティブ作品を整理するための公開アーカイブ索引です。

中心的な目的は、初めて読む人が全体像を見失わないようにすることです。

この索引は、すべてのリポジトリが科学的に証明済みであるとは主張しません。直接冷却、炭素固定回復、惑星循環修復のための、開かれた統合フレームワークとしてリポジトリ群を整理します。

## なぜこの索引が必要か

InchaComisho のリポジトリエコシステムは、気候リスク、直接惑星冷却、海洋・水循環、微生物・炭素固定システム、生態系再生、人工叡智、文明設計、未来コンピューティング、公共コミュニケーションにまたがっています。

マスター地図がない場合、読者は一つのリポジトリだけを見て、それがプロジェクト全体だと誤解する可能性があります。この索引は、慎重な読み方の道筋を示し、次の種類の成果物を区別します。

- 概念フレームワーク
- 仮説
- リスクシナリオ
- 技術提案
- ガバナンス提案
- 公開コミュニケーション
- フィクション / ナラティブ

## 中核命題

このフレームワークは、排出削減は必要だが、それだけで惑星安定性を回復するには不十分かもしれない、という仮説を探究します。

この索引は、排出削減を否定しません。排出削減は気候対応の不可欠な層です。

ここに整理されるリポジトリ群は、直接冷却研究、炭素固定回復、生態系再生、水循環修復、そして自然法則と人工叡智に基づく文明規模のガバナンスという、可能性のある missing layer を探究します。

これらの資料は、単一の介入が準備済み、完成済み、または十分であるという証明ではなく、研究と議論のためのフレームワークとして読むべきものです。

## カテゴリマップ

詳細な地図は [docs/repository-map_ja.md](docs/repository-map_ja.md) を参照してください。

1. **気候 / 直接惑星冷却**  
   直接冷却、地球温暖化の因果構造、補助的な気候対応層に関する概念提案と仮説。

2. **海洋 / 水循環**  
   Ocean Breathing、深海曝気、ナノバブル柱、ミスト遮蔽、都市水循環に関する技術・概念提案。

3. **炭素固定 / 微生物システム**  
   炭素固定崩壊、微生物システム、気候安定性の生物学的基盤に関する仮説とフレームワーク。

4. **砂漠 / 都市 / 生態系再生**  
   砂漠再生、食料生産、循環都市、生態系修復のための概念・技術提案。

5. **人工叡智 / AIガバナンス**  
   AIシステムを長期安定性、人間の監督、生態系継続性、自然法則に整合させるためのガバナンス提案と概念フレームワーク。

6. **文明設計 / 自然法則哲学**  
   持続可能文明、自然法則、和、文明存続に関する哲学的・文明論的・システム設計リポジトリ。

7. **光コンピューティング / 未来技術**  
   光ビーズ量子コンピューティング、多値フォトニック・パラダイムなどの未来技術概念。

8. **フィクション / 公共コミュニケーション**  
   リスクシナリオや修復概念を伝えるためのナラティブ、SF、公開向け作品。

9. **法務 / 社会 / その他**  
   エネルギー、検索、AI価値体系、社会・法務・その他の周辺的または補助的リポジトリ。

5. 高影響の技術提案は、実世界での展開前に、独立レビュー、検証、リスク分析、ガバナンスを必要とする仮説として扱う。
