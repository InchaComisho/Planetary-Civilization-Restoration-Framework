# リポジトリマップ

このマップは、InchaComisho の公開リポジトリをテーマ別・成果物の種類別に整理するものです。

このマップで使うラベル:

- **概念フレームワーク**: 考え方、整理、統合のための構造。
- **仮説**: さらなる証拠を必要とする説明または可能性。
- **リスクシナリオ**: 警告、失敗モード、崩壊経路。
- **技術提案**: システム、機構、実装方向に関する提案。
- **ガバナンス提案**: 意思決定、監督、価値、安全ルールに関する提案。
- **公開コミュニケーション**: 広い読者向けの説明・発信資料。
- **フィクション / ナラティブ**: 物語形式または思弁的なコミュニケーション。

分類はナビゲーションの補助であり、科学的検証を意味するものではありません。

## 分類が不確実なもの

一部のリポジトリは複数カテゴリに属しえます。

- 直接冷却リポジトリは、海洋、砂漠、都市再生と重なる場合があります。
- 人工叡智リポジトリは、文明設計やAIガバナンスと重なります。
- 気候リスクリポジトリは、公開コミュニケーションとしても機能します。
- フィクション/ナラティブ作品は、技術提案ではなくても実在するリスクシナリオを伝える場合があります。

迷う場合は、初めて読む人が安全に発見し、その限界を理解しやすい分類を優先します。
